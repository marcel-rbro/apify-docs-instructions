# apify-docs-instructions

Documentation standards and guidelines for [Apify](https://apify.com) technical documentation. These rules are used by AI assistants and contributors to ensure consistency across platform docs, academy tutorials, and API references.

## What's included

- [`AGENTS.md`](AGENTS.md) - comprehensive documentation guidelines for AI assistants
- [`.claude/rules/`](.claude/rules/) - machine-readable rule files:
  - `writing-style.md` - tone, language, formatting, and grammar rules
  - `content-standards.md` - front matter, headings, admonitions, code examples, and links
  - `terminology.md` - official Apify product names and capitalization
  - `file-organization.md` - naming conventions and directory structure
  - `quality-standards.md` - pre-submission checklist for all content

## Usage

Clone this repo and point your AI coding assistant to the rule files. The `.claude/rules/` directory is automatically loaded by [Claude Code](https://docs.anthropic.com/en/docs/claude-code).
